VirtualMachineX86_64.lbr_enable¶

VirtualMachineX86_64.lbr_enable(cpu_num=None)[source]¶

Enable the Last Branch Record Stack (LBR).

Parameters

cpu_num (int, optional) – The vCPU number to enable the LBR on. If no cpu number is provided, the LBR will be enabled on all vCPUs.
